Чтобы запустить программу, нам понадобится Python. Перед началом установки установите флажок «Добавить Python в PATH». После установки распакуйте архив и запустите файл с именем Перед запуском от имени администратора. После загрузки необходимых библиотек он закроется. Затем перенесите файл с именем T в корневую папку диска C: . Вот и всё, запустите файл EasyUnlock6 от имени администратора, и вы сможете пользоваться программой.
